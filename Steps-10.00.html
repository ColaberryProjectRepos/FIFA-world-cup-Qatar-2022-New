Adding Calendar and Measures Table with measures and columns 10

1. Creating Calendar Table. On Power BI desktop under Modeling click on new table button. Type the Dax formula below in the Dax formula bar, click on the check mark, and save.
calendar = ADDCOLUMNS(CALENDAR(date(YEAR(min(Fifa_world_cup_matches[date])),MONTH(MIN(Fifa_world_cup_matches[date])),DAY(MIN(Fifa_world_cup_matches[date]))),DATE(YEAR(MAX(Fifa_world_cup_matches[date])),MONTH(MAX(Fifa_world_cup_matches[date])),day(max(Fifa_world_cup_matches[date])))),"month number",format( [Date],"MM"),"MONTHE YEAR",FORMAT([Date],"MMMYYYY" ),"MONTH", FORMAT ([Date],"MMM"),"WEAK NUMBER","WEAK-"&WEEKNUM([Date]),"weak",WEEKNUM([Date]),"Day of month",MONTH([Date])&"-"& Format([Date],"dd"),"year quarter",FORMAT([Date],"YYYY") &"/Q"& FORMAT([Date],"Q"),"YEAR",FORMAT([Date],"YYYY"))




2. Creating columns. right click on the calendar table, then click 'new column ' and enter 1 column formula from below and click tick next to it to create that column. click new column  again to create another  column and do the same process until all below columns are created in '_calendar ' table.

columns:

1. Week Start Date = 'calendar'[Date]-WEEKDAY('calendar'[Date],2)+1

2. Week End Date = 'calendar'[Date]-WEEKDAY('calendar'[Date],2)+7

3.  week Duration = FORMAT('calendar'[Week Start Date],"M/D")&"-"& FORMAT('calendar'[Week End Date],"M/D")

4. WEEKRANK = RANKX(ALL('calendar'),'calendar'[Week Start Date],,ASC,Dense)





3. Creating _Measures Table. Go to the "Home" tab and click "Enter Data" and just rename the blank table to "Measures"  and click OK. Create 1 measure at a time.  click on Measures table in the fields section and then table tools ribbon , click 'new measure' and enter 1 measure formula from below and click tick next to it to create that measure. click new measure on the top again to create another measure and do the same process until all below measures are created in '_Measures' table. Some of the measures needs formatting, use the yellow highlighted text to format that measure (it is not part of the DAX)

2. Total Player = COUNT('2022_world_cup_squads'[Player])

 [format- Whole number with 0 places]

3. Total Team = COUNT('2022_world_cup_groups'[Team])

[format- Whole number with 0 places]

4. ALL Team = CALCULATE([Total Team],ALL('2022_world_cup_groups'[Team],'2022_world_cup_groups'[Continent]))

5. Total world cups = COUNTROWS(world_cups)

6. Total goals =

    SUM(Fifa_world_cup_matches[number of goals team1]))

var b = 
CALCULATE(
    SUM('Fifa_world_cup_matches (2)'[number of goals team2]))

7. goal this week = CALCULATE([Total goals],FILTER(ALL('calendar'),'calendar'[WEEKRANK]=MAX('calendar'[WEEKRANK])))

[format- Whole number with 0 places]

8. goal last week = CALCULATE([Total goals],FILTER(ALL('calendar'),'calendar'[WEEKRANK]=MAX('calendar'[WEEKRANK])-1))

[format- Whole number with 0 places]

 

9.  Goal WOW % = CALCULATE(DIVIDE([goal this week]-[goal last week],[goal last week],0)/[ALL Team])+0

[format-decimal with 2 places]

8. Color Arrow goals = IF([Arrow Goals] = UNICHAR(9660),"Red","Green")
