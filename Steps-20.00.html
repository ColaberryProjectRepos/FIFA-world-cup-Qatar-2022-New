  1. Click on an empty space on the canvas, and from the visualizations pane select Line and stacked Column Chart.
  2. Fill up fields X- axis = Week Duration ,Day of month ; Column Y-Axis = Total Goals; Line Y-Axis =Goals WOW %,Goals Preventions WOW %, Cards WOW%, Shot Attempts WOW % ;  Tooltips =Goals WOW %,Goals Preventions WOW %, Cards WOW%, Shot Attempts WOW %.
  3. In Format Visual: X-axis: Values: Font=Segoe UI (8),Bold,color=black, Title - off  Y-axis: Values: Font=Segoe UI (9),Bold,color=black, Title - off . Secondary Y-axis: Values: Font=Segoe UI (9),Bold,color=black. Legend: Position- Top left, Style Marker (Circle default), Text:  Font=Segoe UI (8),Bold,color=black. Gridlines: on,Style= Dotted,Color=#c8c6c4. Columns Color= #e8d166.              Lines: line style: solid, join type: round, stroke width: 3px, Colors: Shot Attempts WOW %=#3654E6, Cards WOW%=#AD2828, Goals Preventions WOW %=#3a2761, Goals WOW %=#666666.
  4. For Format  General: Properties, Size, Height=321, Width=602, Position, Horizontal= 227, Vertical=248. Title=On, Text=Weeks, Font=Segoe UI(17),Bold, Text color= #FFFFFF, Background color=#a1343c , Horizontal alignment=Center, Effects: Background=On, Background color= #FFFFFF,, Transparency=0%. visual border :color = whit, rounder corners = 0px.
  5. Sort axis : Week Duration, sort ascending.
